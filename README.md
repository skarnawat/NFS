# How to Use:

1. You can either `git clone https://github.com/skarnawat/NFS.git` or just Save the content of configure_gcve_storage.sh script file above as a .sh file (e.g., configure_gcve_storage.sh).
2. Make it executable: chmod +x configure_gcve_storage.sh.
3. Run the script: ./configure_gcve_storage.sh
